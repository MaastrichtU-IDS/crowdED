CrowdED: Guideline for designing optimal crowdsourcing experiments
====

[CrowdApp](https://pedrohserrano.shinyapps.io/crowdapp/) Beta

## Usage

Clone this repo

        git clone https://github.com/pedrohserrano/crowdED.git
        cd crowdED

Install crowdED

        pip install crowdED
